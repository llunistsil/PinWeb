# Документация API

## Описание

REST API для системы управления задачами с поддержкой расписания и учёта времени.

## Разделы

- [Модели данных](models.md)
- [Аутентификация](auth.md)
- [Тестирование](tests.md)

## API Reference

| Endpoint | Описание |
|----------|----------|
| `/users` | Пользователи |
| `/users/me` | Профиль текущего пользователя |
| `/tasks` | Задачи |
| `/priorities` | Приоритеты |
| `/schedules` | Расписание |
| `/schedule_tasks` | Связь задач с расписанием |
| `/time_entries` | Учёт времени |
| `/notifications` | Уведомления |
