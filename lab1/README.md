# Лабораторная работа №1 по веб-программированию

## Описание

Проект представляет собой backend-приложение на FastAPI с реализацией системы управления задачами и пользователями.

## Основной функционал

- Регистрация и авторизация пользователей
- Аутентификация через JWT-токены
- Управление задачами (создание, редактирование, удаление)
- Работа с расписанием задач
- Учёт времени (time entries)
- Система уведомлений
- Приоритеты задач
