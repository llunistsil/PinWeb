# Лабораторная работа №2

## Описание

Лабораторная работа посвящена изучению многозадачности в Python.

## Задачи

1. Изучение модуля `multiprocessing` для работы с процессами
2. Изучение модуля `threading` для работы с потоками
3. Изучение модуля `asyncio` для асинхронного программирования

## Структура

- `tasks/task1/` — базовые примеры многозадачности
- `tasks/task2/` — практическое применение (парсинг и сохранение данных)
